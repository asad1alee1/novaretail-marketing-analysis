# NovaRetail – Customer Segmentation & Marketing Effectiveness

## Overview
Full marketing analytics pipeline built in Python analyzing customer 
behavior, segmentation, and marketing channel performance for a 
retail business.

## What this project covers
- **RFM Analysis** — Recency, Frequency, Monetary profiling across customers
- **K-means Clustering** — Segmented customers into 4 groups: Champions, 
  Engaged Potentials, Discount Hunters, and Dormant
- **Acquisition Channel ROI** — Compared CLV and spend across marketing channels
- **Email Marketing Effectiveness** — Analyzed email opens/clicks vs CLV 
  across quintiles
- **Discount ROI Analysis** — Measured whether discount users generate 
  higher long-term value
- **Data Visualizations** — EDA dashboards, segment profiles, scatter plots, 
  correlation heatmaps

## Tools
Python · pandas · numpy · scikit-learn · seaborn · matplotlib

## Key Findings
- Champions segment generated highest CLV despite lowest discount usage
- Email engagement showed positive correlation with CLV across all quintiles
- Acquisition channel significantly predicted long-term customer value
- Discount Hunters showed high frequency but lower CLV than Champions
